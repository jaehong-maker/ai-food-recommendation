# AI 음식 추천 (먹을래 말래)

## 주요 기능
- 사용자 맞춤형 음식 추천 (시간대, 취향, 건강)
- 랜덤 메뉴 추천
- 예산/위치/날씨 기반 추천
- 피드백 기반 AI 학습
